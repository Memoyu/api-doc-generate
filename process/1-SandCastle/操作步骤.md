#### 2. 使用cmd

> 前提：需要使用dotnet，依赖.net framework 4.8 以及安装SHFB

##### 命令

使用如下命令进行构建

```shell
# 命令格式：msbuild [.shfbproj路径]
# 默认生成文件与.shfbproj同一目录下的Help文件夹中
dotnet build D:\\Builder\\Builder.Sample.shfbproj
```
